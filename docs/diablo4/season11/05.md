title: 5.어떤 던전을 돌아야 하고 어떤 재료를 얻을 수 있을까?
tags: [디아블로, 디아블로시즌11]
---

**⇒ 공통적으로 던전은 가능한 제일 높은 단계의 던전을 도는 것이 기본이다.**

- <span className="red">잘 모르겠다면</span> 지옥 물결을 가능한 높은 난이도에서 돌아주고 상자를 열어주고 <span className="green">지옥 물결 내에 있는 두리엘</span>을 잡던가 <span className="green">아즈모단</span>을 잡으러 간다.
- <span className="red">마법 부여나 명품화 재료</span>가 필요하다면 <span className="green">지옥 물결</span>을 돌고 상자를 열어 나오는 아이템들을 갈아준다.
- <span className="red">강해지고 싶다면</span> <span className="green">세리가 마을에서 명공의 나락</span>을 클리어 할 수 있는 난이도로 돌고 문양을 업그레이드 해준다.
- <span className="red">옵두사이트</span>가 필요하다면 <span className="green">군세</span>를 돌거나 <span className="green">악몽의 인장에서 보물의 틈새, 호라드림 금고실</span> 효과가 있는 인장을 돌아준다.
- 담금질 리셋을 해서 전설 옵션을 띄우고 싶다면 <span className="red">복원의 두루마리</span>를 얻기 위해 <span className="green">군세나 월드 보스, 군단 이벤트</span>를 돌아준다.
- 명품화, 마법 부유 등을 위해 <span className="red">잊힌 영혼</span>이 필요하면 <span className="green">쿠라스트 지하도시, 지옥 물결, 속삭임의 나무, 악몽의 인장에서 보물의 틈새</span>를 돌아준다.
- <span className="red">레벨업</span>을 하고 싶다면 <span className="green">지옥 물결에서 두리엘</span>을 잡아준다.
- 정화된 아이템 드랍
    - 정화된 아즈모단 : 맵 우측 하단 아즈모단
    - 정화된 벨리알 : 나락
    - 정화된 두리엘 : 지옥 물결
    - 정화된 안다리엘 : 쿠라스트 지하도시(조화의 공물(확인 된 공물), 다른 것도 아마도 가능)
